# 📱 React Native: Primeiro Desafio Expo

Um projeto inicial e prático desenvolvido com **React Native** e **Expo** para replicar uma interface de boas-vindas. O foco foi o aprendizado de estilização básica e layout.

### 🛠️ O que foi praticado:
* **Flexbox:** Centralização de elementos na tela.
* **Componentes Core:** Uso de `View`, `Text` e `StatusBar`.
* **Estilização:** Aplicação de cores de fundo, bordas arredondadas e espaçamentos internos.
* **Inline Styles:** Prática direta de propriedades CSS aplicadas ao mobile.

### 🚀 Como executar:
1. Clone o repositório.
2. Execute `npm install`.
3. Inicie com `npx expo start`.
4. Abra o app **Expo Go** no celular e escaneie o QR Code.
